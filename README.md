## 📂 Featured Projects

### 🗺️ [Spatial Grid Analysis](https://github.com/AleksiAu/spatial-grid-analysis)
**How is electricity demand distributed across the network area?**
Built 1 km² grid-based consumption maps, customer segmentation, and asset age visualization using GeoPandas + Folium. Animated time-slider shows how demand evolves year-over-year.

### 🌡️ [Load–Weather Analysis](https://github.com/AleksiAu/load-weather-analysis)
**How weather-sensitive is our network load, and what are the structural demand trends?**
Regression analysis quantifying MW/°C sensitivity by customer group. Temperature-corrected load series for noise-free trend identification. Seasonal profiles and December cold-weather analysis.

### 🤖 [Load Forecasting with LSTM](https://github.com/AleksiAu/load-forecasting-ml)
**Can we predict hourly network load from historical patterns and weather?**
End-to-end ML pipeline: DuckDB feature extraction → LSTM neural network → evaluation against persistence baseline. Includes 15+ per-customer behavioral features computed in a single SQL query.
